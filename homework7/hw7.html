Homework 7 - Object Model and Event Model

  • Part 1
  • Part 2

Part 1

Allows users to apply various styles (background color, text formatting, and font size) to a content area based on their selections from a set of form elements. The applyStyles() function checks the state of the background color radio buttons, text formatting checkboxes (bold, italic, underline), and the selected font size to update the content area's styles. Event listeners are used to trigger the applyStyles() function whenever a user changes any of the form elements. The function is also called initially to apply default styles.

Part 2

Manages the display of dropdown menus in a navigation bar. It listens for mouseover and mouseout events on each navigation link (navLinks). When a user hovers over a link, the associated dropdown menu (if present) is shown by setting its display style to 'block'. When the mouse leaves the link, the dropdown menu is hidden by setting the display style to 'none'. This enables a simple hover-based dropdown functionality.
