Part 2

This part defines a function, searchCharacter(), that searches for a specific character in a given text input and displays the results. It retrieves the text content and the character to search, ensuring the character is a single letter. If the input is invalid, an error message is shown. Otherwise, it converts both inputs to lowercase for case-insensitive comparison and counts how many times the character appears in the text. If the character is found, the count is displayed on the webpage. If not, a new browser window opens with a message stating that the character was not found, and a button is provided to close the window. Additionally, the main output area on the webpage is cleared when the character is not found.

Part 3

This part uses jQuery to format a phone number input field and provides functions to validate and process the input. When the page is ready, a jQuery mask is applied to the phone input field to enforce the format (999) 999-9999. The processPhoneNumber() function checks if the input matches this format using a regular expression. If valid, it extracts the area code, first three digits, and last four digits, populating respective fields. If the input is invalid, it clears these fields and displays an error message prompting the user to enter the correct format. The clearForm() function resets the phone input field, extracted values, and any displayed messages, allowing the form to be reused.
