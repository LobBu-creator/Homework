Part 1

part 2

Calculates the sum, average, product, smallest, and largest values from three user-input numbers. It validates the inputs to ensure they are numbers and displays an error message if any input is invalid. Results are shown in a textarea with formatted explanations. A clear button resets all input fields and the result display. Event listeners handle button clicks, and event.preventDefault() prevents page refresh on form submission.
