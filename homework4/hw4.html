Homework 4 - Looping Statements

  • Part 1
  • Part 2
  • Extra Credit

Part 1

Calculates products and sums using loops and adds a draggable element with jQuery. A for loop computes the product (5 * 9 * 13 * 17 * 21 * 25) and sum (5 + 9 + 13 + 17 + 21 + 25), displaying results in for-loop-result. A while loop calculates the product (3 * 6 * 9 * 12 * 15 * 18) and sum (3 + 6 + 9 + 12 + 15 + 18), displaying results in while-loop-result. Lastly, jQuery's .draggable() makes an element with the ID draggable movable.

Part 2

This jQuery script dynamically generates compound interest tables for multiple interest rates. It starts with a principal amount of $1,000 and calculates the compound interest for 5 years at rates of 5%, 6%, and 7%. For each rate, a table is created showing the year, the compounded amount, and the interest rate. These tables are styled with white headings and appended to an HTML element with the ID tables. This demonstrates dynamic content creation and looping through arrays and nested data using jQuery.

Extra

Generates a square-shaped pattern of asterisks (*) based on user input. The user enters a size (between 2 and 10), and the program validates the input. If valid, it creates a hollow square pattern with * for the border and spaces inside. The pattern is displayed in an HTML element with the ID result. A separate function clears the input and output when called.
