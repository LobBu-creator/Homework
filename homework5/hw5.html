Homework 5 - Arrays

  • Part 1
  • Part 2
  • Part 3

Part 1

Handles form submission and validation. When the user submits the form, it collects the full name, selected age group, chosen browsers, and preferred movie type. If any required fields are missing, it displays an error message. If all fields are filled, it shows a success message in green. A separate function clears the form and resets the result display to its default state.

Part 2

Defines two functions to open URLs in a new tab. The first function, triggered by the onchange event of menu1, opens the selected URL from the first dropdown. The second function, triggered by the onclick event of menu2, does the same for the second dropdown. Both functions ensure a URL is selected before opening it.

Part 3

Allows users to input a U.S. state abbreviation or name and retrieve information about the state, such as its abbreviation, name, capital, and population. The getStateInfo() function compares the user input with the state data and displays the corresponding details if a match is found. If no match is found, it informs the user that information is only available for a predefined list of states. The clearForm() function resets the input field and output area.
